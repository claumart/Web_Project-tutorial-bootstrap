NAVS

Los navs son elementod de navegación. Se puede hacer uso de ellos a traves de la clase .nav Existen varios tipos de componentes nav

Navegación con Pestañas

para acceder a esta función utiliza la clase .nav-tabs de la siguiente manera:

<ul class="nav nav-tabs">
	<li role="presentation" class="active"><a href="#">Home</a></li>
	<li role="presentation"><a href="#">Profile</a></li>
	<li role="presentation"><a href="#">Messages</a></li>
</ul>

Resultado

  • Home
  • Profile
  • Messages


Navegación con Botones

Aplicando la clase .nav-pills al mismo código HTML del ejemplo anterior, los enlaces se muestran como botones ("pills" en inglés) en vez de como pestañas. Ejemplo: 

<ul class="nav nav-pills">
  <li role="presentation" class="active"><a href="#">Home</a></li>
  <li role="presentation"><a href="#">Profile</a></li>
  <li role="presentation"><a href="#">Messages</a></li>
</ul>

Resultado

  • Home
  • Profile
  • Messages
